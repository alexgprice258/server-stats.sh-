# Server Performance Stats

A Bash script that prints basic server performance stats:
- Total CPU usage
- Memory usage (used/free + %)
- Disk usage (used/free + %)
- Top 5 processes by CPU and memory

## Run
```bash
chmod +x server-stats.sh
./server-stats.sh
